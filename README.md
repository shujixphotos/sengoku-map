# 戦国時代 勢力変遷マップ

日本列島を旧国（甲斐・信濃・尾張…）単位で塗り分け、年代スライダーで勢力の移り変わりを見るWebコンテンツです。
外部ライブラリ・ビルド環境なし。**ファイルを開くだけで動きます。**

---

## 1. 起動方法

### いちばん簡単な方法
`sengoku-map.html`（1ファイル版）をダブルクリックしてブラウザで開くだけです。

### 開発用（推奨・分割版）
フォルダごと保存して `index.html` を開きます。

```
index.html
css/style.css
js/geo.js     … 海岸線と旧国の代表点、地図ポリゴンの生成
js/kamon.js   … 家紋（簡略図案）
js/data.js    … 勢力・年代データ（ここだけ触れば内容が増やせます）
js/app.js     … 画面の組み立てと操作
```

`file://` のままで動きますが、ローカルサーバーを立てても構いません。

```bash
cd sengoku-map
python3 -m http.server 8000
# → http://localhost:8000
```

## 2. スマートフォンでの確認方法

1. PCとスマホを同じWi-Fiにつなぐ
2. PCで `python3 -m http.server 8000` を実行
3. PCのIPアドレスを調べる（`ipconfig` / `ifconfig` / `ip a`）
4. スマホのブラウザで `http://192.168.x.x:8000` を開く

サーバーを立てられない場合は、1ファイル版 `sengoku-map.html` をスマホに送って開けば単体で動きます。
（Chrome DevTools のデバイスモードでも確認できますが、ピンチ操作は実機が確実です）

## 3. 操作

| 操作 | 動き |
|---|---|
| 上部スライダー / ‹ › / 年号タップ | 年代を切り替え |
| 地図の色つき領域をタップ | その勢力の詳細を表示（スマホ=ボトムシート、PC=右パネル） |
| ドラッグ | 地図を移動 |
| ピンチイン・アウト / ホイール | 拡大縮小 |
| ダブルタップ | 拡大 |
| 「全体」ボタン | 初期表示に戻す |
| ← → キー | 年代の前後移動 |

## 4. データの追加・変更方法

### 年代を増やす
`js/data.js` の `SENGOKU_YEARS` に要素を足すだけです。スライダーの目盛りは自動で増えます。
1年単位にしたければ、年ごとのオブジェクトを並べるだけで動きます（配列の順=スライダーの順）。

```js
{
  year: 1585, era: '天正13年', note: '四国攻めの頃',
  owners: {            // 書かなかった国は自動で 'other'（その他の諸勢力）
    kai:'tokugawa', shinano:'tokugawa', owari:'toyotomi', /* ... */
  },
  clans: {
    toyotomi: {
      head:'羽柴秀吉',
      castle:{ name:'大坂城', lon:135.50, lat:34.69 },  // 経度・緯度
      lands:'畿内・北陸・中国東部',
      retainers:['蜂須賀正勝','黒田官兵衛'],
      power:5,                       // 1〜5（★の数）
      events:['四国を平定','関白に就任'],
      desc:'解説文'
    },
    other: { head:'—', castle:null, lands:'各地', retainers:[], power:1, events:['…'], desc:'…' }
  }
}
```

> `owners` に出てくる勢力は、その年の `clans` にも必ず定義してください（`other` を含む）。

### 勢力を増やす
`SENGOKU_CLANS` に1行足します。

```js
ukita: { name:'宇喜多家', color:'#5f7f8f', kamon:'other' },
```

`kamon` は `js/kamon.js` のキー名。新しい家紋は `kamon.js` の `S` に
100×100座標のSVG図形を足せば使えます。

### 旧国を増やす・動かす
`js/geo.js` の `PROVINCES` に「代表点（経度・緯度）」を1行足すだけで、
地図のポリゴンが自動で作り直されます（隣の国が自動的に譲ります）。

```js
{ id:'oki', name:'隠岐', island:'oki', lon:133.30, lat:36.20 },
```

新しい島を足す場合は `COAST` に海岸線ポリゴン（[経度,緯度] の配列）を追加してください。

### 武将画像を入れる
現在は家紋を使った仮表示です。`js/app.js` の `renderPanel()` 内 `.portrait` を
`<img src="img/xxx.jpg">` に置き換えれば実画像になります（データに `portrait:'img/katsuyori.jpg'` を持たせる想定）。

## 5. 地図データについて（重要）

- 地図は **正式な旧国GeoJSONではなく、簡略化した自動生成**です。
  「粗い海岸線ポリゴン」＋「旧国の代表点」から、最寄りの代表点で陸地を塗り分け（ボロノイ相当）、
  輪郭を抽出・平滑化してSVGパスを作っています。国境線は史実の国境と一致しません。
- 正式データへ差し替える場合は、`GEO.build()` が
  `{ paths:{国ID: SVGパス文字列}, coast:…, ezo:… }` を返す契約だけ守れば、
  `app.js` 側は無改修で動きます。
- 便宜上の簡略化:
  - 陸奥が広大なため、`mutsu`（陸奥＝現在の岩手・宮城以北）と `nanou`（岩代・磐城＝南奥州）に分割しています。
  - 蝦夷地・隠岐・対馬などは色分け対象外（蝦夷地は装飾表示のみ）。
  - 1550年の織田は信秀、1600年の豊臣は「西軍寄りの大名領をまとめた表示」など、
    1色1家に収めるための簡略化があります。詳細は各年の解説文に記載。

## 6. セルフレビュー結果

- [x] 年代切替：スライダー / ‹› / 年号タップの3経路で切替。国の色は0.55秒でクロスフェード、勢力外形線も再計算される
- [x] 勢力エリアのタップ：`<path data-prov>` に対するタップ判定（ドラッグ10px未満かつ500ms未満のときだけ選択）
- [x] 詳細表示：当主・家紋・本拠地・主な領土・主な家臣・強さ・その年の出来事・解説・領国一覧
- [x] スマホの拡大移動：Pointer Eventsでドラッグ／2本指ピンチ／ダブルタップ拡大（0.7〜8倍）
- [x] 画面外へのはみ出し：`html,body{height:100dvh;overflow:hidden}` + flexレイアウト。パン範囲もクランプ済み
- [x] レスポンシブ：900px以下で右パネル→ボトムシート、勢力一覧→引き出しに切替
- [x] データ追加のしやすさ：地図生成・画面処理・データが3ファイルに分離。年代追加は配列に1要素足すだけ
- [x] 文字の大きさ：地図ラベルはSVGの外（HTML）に置き、拡大しても一定サイズ。重なったラベルは自動で間引き、拡大すると現れる
- [x] データ整合性：全年代で「owners に出る勢力＝clans に定義あり」を検証済み
- 既知の制約：初回描画時に地図ポリゴンを生成するため、低速端末で0.3秒ほど時間がかかります（以降はキャッシュ）。
